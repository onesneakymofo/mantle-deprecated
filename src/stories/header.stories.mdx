### Sizes
Headers can be sized by adding the `size` tag to the component. Sizes come in
`xs`, `sm`, `md`, `lg`, `xl`, `2xl`.  No size specified is our default size (`md`).
<Preview>
  <Header size="2xl">This is a header!</Header>
  <Header size="xl">This is a header!</Header>
  <Header size="lg">This is a header!</Header>
  <Header size="md">This is a header!</Header>
  <Header size="sm">This is a header!</Header>
  <Header size="xs">This is a header!</Header>
</Preview>
