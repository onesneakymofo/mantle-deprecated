### Sizes
Avatars can be sized by adding the `size` tag to the component. Sizes come in
`xs`, `sm`, `md`, `lg`, `xl`.  No size specified is our default size (`sm`).

### Avatars (Name Only)
If no source is provided or the source is broken and a name is passed to the Avatar,
we default to the first letters of the first and last name as a placeholder.
