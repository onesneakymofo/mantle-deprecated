# Grid

### Usage
<Preview>
  <Grid className="grid-rows-3 grid-flow-col gap-4 text-white">
    <GridItem className="row-start-3 row-span-1 bg-blue-500 p-4">I'm</GridItem>
    <GridItem className="row-start-2 row-span-1 bg-red-500 p-4">a</GridItem>
    <GridItem className="row-start-1 row-span-1 bg-green-500 p-4">grid!</GridItem>
  </Grid>
</Preview>

### Information
In order to maximize CSS Grid's potential, `Grid` and `GridItem` only accept the `className` tag.  More information on what classes you can pass to both components be found on <a href="https://https://tailwindcss.com/" target="_blank">Tailwind CSS's website</a>. For more information on CSS Grid, check out Morten Rand-Hendriksen's <a href="https://www.youtube.com/watch?v=txZq7Laz7_4" target="_blank">CSS Grid Changes Everything (About Web Layouts)</a>.
